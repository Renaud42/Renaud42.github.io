Présentation 🔗

Je m'appelle Renaud, je suis né le 10 décembre et je suis Français.

Un jour, j'ai souhaité créer un jeu, du coup je suis tombé dans le monde de l'informatique et surtout dans le domaine du développement

Je concevais de petites applications telles un petit bloc-note ou un navigateur web avancé.

J'ai plus tard appris d'autres langages comme les langages du web, le Java et d'autres...

ADMIN SUR ALPHARP

AlphaRP est un serveur Minecraft RolePlay & Mini-jeux de type Garry's Mod avec launcher.

Je suis Administrateur dessus, je gère leur site web, leur Discord et leur launcher.

Les langages utilisés sont côté site :

  • Le HTML (Langage de balisage hypertexte)
  • Le CSS (feuilles de style en cascade)
  • Le JS (JavaScript) avec notamment l'API de MC-API.net et l'outil Particles.js.
  • Un tout petit peu de PHP (Préprocesseur Hypertexte) juste pour certaines redirections et pour des include pour clarifier le code. Je ne l'ai pas trop utilisé dans le sens où il n'y a pas de système de connexion ou de pages qui nécéssitent des cookies.

Et côté launcher, uniquement le Java avec les librairies de Litarvan
