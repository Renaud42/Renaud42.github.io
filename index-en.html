ADMIN AT ALPHARP

AlphaRP is a French RolePlay & Minigames Minecraft modded server like Garry's Mod.

I'm administrator, and I manage the website, the Discord and the custom game launcher.

The website languages are :

  • The HTML (HyperText Markup Language)
  • The CSS (Cascading StyleSheets)
  • The JS (JavaScript) noting with the MC-API.net API and the Particles.js tool.
